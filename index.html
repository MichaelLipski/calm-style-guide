Common CSS Properties

Property What It's Used For Example
Color Changing the foreground color such as text color: #000000;
Font Size Changing the size of the displayed text font-size: 14px;
Background Color Changes the background color of an element background-color: #FFFFFF;
Height/Width Adjusts the height and width of an element height: 25px; width: 30px;
Border Assigns border properties to an element border: 5px solid #FF6DBA;
Border Radius Rounds out the corners of an element border-radius: 25px;
